# 🚗 CRAIC 蓝牙遥控全地形小车

> **CRAIC 全地形小车挑战赛 · 省赛入围作品**

基于 Arduino 的蓝牙遥控全地形小车，通过单字符蓝牙指令实现远程控制，集成三自由度机械臂、OLED 实时状态显示和 GM65 二维码扫描功能。

---

## ✨ 功能特性

- 🎮 **蓝牙遥控** — 前进 / 后退 / 原地左右转 / 微调弧线转向，共 8 种运动模式
- ⚡ **三级调速** — 逐级加减速（步进 20）+ 一键极速模式（PWM 245）
- 🦾 **三自由度机械臂** — 大臂摆动 + 小臂俯仰 + 夹爪开合，两个预设抓取位置
- 🖥️ **OLED 实时显示** — 运动状态 / 速度 / 机械臂三轴角度实时反馈（128×64 SSD1306）
- 📷 **二维码扫描** — 集成 GM65 模块，扫描结果串口回传并显示于 OLED
- 🛡️ **断连安全保护** — 心跳超时机制，2 秒无指令自动停车，防止失控

```

├── arm.ino                         # 机械臂三轴舵机控制
├── OLed_Show.ino                   # OLED 显示模块

```

## 📡 蓝牙指令表

### 运动控制

| 指令 | 功能 | 实现方式 |
|------|------|----------|
| `F` | 前进 | 左右轮同速正转（currentSpeed） |
| `B` | 后退 | 左右轮同速反转（currentSpeed） |
| `L` | 左转 | 左轮反转 + 右轮正转，原地旋转（TURN_SPEED=180） |
| `R` | 右转 | 左轮正转 + 右轮反转，原地旋转（TURN_SPEED=180） |
| `J` | 微左 | 仅右轮驱动，弧线左转 |
| `K` | 微右 | 仅左轮驱动，弧线右转 |
| `S` | 停止 | 所有电机 PWM 归零 |
| `E` | 刹车 | 四路满占空比制动 100ms 后关闭 |

### 速度控制

| 指令 | 功能 |
|------|------|
| `W` | 加速（+20，上限 255） |
| `Z` | 减速（-20，下限 80） |
| `G` | 一键极速（PWM 245） |

### 机械臂控制

| 指令 | 功能 |
|------|------|
| `O` | 夹爪张开（15°） |
| `C` | 夹爪夹紧（55°） |
| `H` | 机械臂归位（110° / 35° / 40°） |
| `1` | 预设位置 A（大臂 70°，小臂 70°） |
| `2` | 预设位置 B（大臂 180°，小臂 175°） |

### 辅助功能

| 指令 | 功能 |
|------|------|
| `Q` | 触发二维码扫描，结果显示于 OLED 并串口回传 |
| `I` | 查询系统状态（当前速度 + 三轴角度） |
| `?` | 打印帮助信息 |

## 🧠 技术亮点

**ServoTimer2 自定义舵机库**
标准 Arduino `Servo` 库使用 Timer1，与 D9/D10 的 PWM 电机控制产生冲突。本项目采用基于 Timer2 溢出中断的 `ServoTimer2` 库，通过精确计算计数器值和余数实现 750~2250μs 脉冲控制，彻底解决了舵机与电机 PWM 的资源争用问题。

**心跳超时安全机制**
主循环以 10ms 为周期检测 `millis() - lastCmdTime`，超过 2000ms 自动停车。2 秒阈值既能快速响应断连，又不会因蓝牙跳频导致的短暂延迟误触发，同时采用停止而非刹车策略，避免急停导致物体翻落。

**`F()` 宏节省 SRAM**
所有串口和 OLED 输出的字符串字面量均用 `F()` 宏存储于 Flash，为 ATmega328P 有限的 2KB SRAM 留出更多空间给运行时变量。

## 🚀 快速开始

1. 克隆本仓库

   ```bash
   git clone https://github.com/yuanquanlink/CRAIC-blutooth-car.git
   ```

2. 用 Arduino IDE 打开 `arm` 文件夹

3. 安装依赖库：Arduino IDE → 工具 → 管理库 → 搜索安装 `Adafruit SSD1306` 和 `Adafruit GFX`

4. 选择对应开发板（Arduino Uno / Mega）和串口，上传代码

5. 用手机安装蓝牙串口 App（如 **Serial Bluetooth Terminal**），配对 HC-05，发送单字符指令控制小车
